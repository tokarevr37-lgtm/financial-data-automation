# Financial Data Automation

A Python automation tool for collecting financial market data from API sources and exporting clean structured reports.

## Output Preview

## What this project does

This project demonstrates a practical API automation workflow for collecting structured financial data and exporting client-ready reports in CSV and Excel formats.

Collected data includes:

- Asset symbol
- Current price
- Daily change percentage
- Day high
- Day low
- Trading volume

## Features

- API data collection
- CSV export
- Excel export with formatting
- Structured reporting workflow
- Error handling
- Clean reusable automation logic

## Tech Stack

- Python
- Requests
- OpenPyXL
- CSV
